OGC Sensor Model Language 1.0.1 Executable Test Suite

Overview

This test suite is based on the following OGC specification: * Sensor Model Language Documents, version 1.0.1 OGC 07-000(http://portal.opengeospatial.org/files/?artifact_id=21273)

What is tested

The test performs a test against the SensorML schema. The SensorML can be a stand alone file or embedded inside a swes:DescribeSensorResponse element.
